**Link to backend repo**: https://github.com/ayushjrathod/Advista_api

Advista is a solution that identifies user pain points and triggers from multiple data sources like Google, YouTube, Reddit, Quora, blogs, forums, social media, and app reviews. It analyzes competitor ads and strategies to uncover high-performing hooks, CTAs, and content formats based on the target audience, ad objectives, and ad formats.
